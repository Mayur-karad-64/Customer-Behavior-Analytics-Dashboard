# Customer-Behavior-Analytics-Dashboard
Built an ETL pipeline to analyze customer shopping behavior using Python (Pandas), PostgreSQL, and Power BI with live database connectivity.

# 📊 Customer Behavior Analytics Dashboard  
### End-to-End Data Pipeline using Python, PostgreSQL & Power BI

---

## 📌 Project Overview

This project builds a complete end-to-end data analytics pipeline to analyze customer shopping behavior and generate business insights.

The workflow includes:
- Data Cleaning & Transformation using Python (Pandas)
- Feature Engineering
- Data Storage in PostgreSQL
- Interactive Dashboard creation in Power BI (connected directly to PostgreSQL)

This project demonstrates real-world data pipeline architecture used in industry.

---

## 🎯 Problem Statement

Businesses need insights into:
- Customer segmentation
- Revenue distribution across categories
- Subscription impact on sales
- Age-based purchasing behavior

This project transforms raw customer data into actionable business intelligence.

---

## 🛠 Tech Stack

- Python (Pandas, SQLAlchemy)
- PostgreSQL
- Power BI

## 🧹 Data Cleaning & Transformation

- Handled missing values using category-wise median imputation
- Standardized column names for SQL compatibility
- Created age groups using quartile-based segmentation
- Converted purchase frequency into numeric days
- Removed redundant columns
- Loaded structured data into PostgreSQL

---

## 📊 Dashboard KPIs

- Total Customers
- Average Purchase Amount
- Average Review Rating
- Revenue by Category
- Sales by Age Group
- Subscription Distribution

---

## 💡 Key Insights

- Clothing category generates the highest revenue.
- Young Adults contribute significantly to total sales.
- Majority customers are non-subscribers.
- Subscription status influences purchasing patterns.
